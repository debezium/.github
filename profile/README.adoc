Debezium is an open-source distributed platform designed for Change Data Capture (CDC), providing applications with an industry-standard way to respond to real-time database changes.

See our https://debezium.io[website] for more details about the project.

=== Getting started with Debezium?

Check out our https://debezium.io/documentation/reference/stable/tutorial.html[tutorial] to learn how to set up and deploy Debezium.

=== Need help?

If you want to get in touch with us, you can use:

* https://debezium.zulipchat.com/#narrow/stream/302529-users[Zulip chat]
* https://groups.google.com/forum/#!forum/debezium[Mailing list]

=== Want to Contribute?

You can contribute by:

* Raising an issue in our https://github.com/debezium/dbz/issues[issue tracker].
* Fixing issues by opening pull requests.
* Improving the https://debezium.io/documentation/reference/stable[documentation], source located https://github.com/debezium/debezium/tree/main/documentation[here].
* Talking about and raising awareness about Debezium.

Have other questions about contributing, see our https://github.com/debezium/debezium/blob/main/CONTRIBUTE.md[Contributing Guide] for more details.
